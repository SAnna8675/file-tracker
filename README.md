# File Tracker - VS Code Extension

Расширение для Visual Studio Code, которое отслеживает файлы (например, книги) и отправляет вам уведомление, если файл не был проверен в течение 30 дней.

## Возможности

-   Ежедневная проверка файлов по расписанию.
-   Умные уведомления: Уведомление приходит каждые 30 дней.
-   Система ошибок: Плагин сообщает о проблемах (файл не найден, некорректная дата и др.).
-   Визуальная история: С каждым уведомлением добавляется знак "&" (до 5 знаков).

## Установка

### Установка из VSIX (для тестирования)

1. Соберите пакет расширения: vsce package
2. В VS Code откройте панель расширений (Ctrl+Shift+X).
3. Нажмите на "..." и выберите "Install from VSIX..."
4. Выберите файл file-tracker-1.0.0.vsix

## Использование

1.  После установки расширение активируется автоматически.
2.  Создайте файл file-tracker-data.json в вашей домашней директории (~/file-tracker-data.json).
3.  Добавьте в него информацию о файлах, которые хотите отслеживать.

### Пример файла данных

`json
{
    "trackedFiles": [
        {
            "path": "/full/path/to/your/file.pdf",
            "fileName": "My Book.pdf",
            "addedDate": 1704067200000,
            "lastNotificationDate": 1704067200000
        }
    ]
}
